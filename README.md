## 功能特点
1. 提供CNC铣床G代码快速输入及参数说明功能
2. 支持X/Y/Z/A/B/C/U/V/W轴向小数点缺失检测
3. 通过颜色标注重要性等级，红色表示关键问题（如缺少小数点）

## 快捷命令
快捷命令 | 功能说明
:------:|:------
Q      | 起始模板
G00~G99| 插入G代码（G需大写），输入G后显示代码定义

## 使用方法
安装启用插件后，当文件语言设为`gcode`或保存为`.nc`格式时自动激活。输入`G`可调出帮助向导。

## 安装
